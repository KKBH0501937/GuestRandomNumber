# Jeu du nombre à deviner

Projet Java réalisé dans le cadre du cours Projet Informatique II

## Description

L'ordinateur génère un nombre aléatoire entre 1 et 100. Le joueur a 10 essais pour le deviner. Après chaque tentative, le programme indique si le nombre proposé est trop grand ou trop petit.

## Structure du projet

- GuestRandomNumber.java : classe qui gère la logique du jeu (nombre secret, tentatives, vérification)
- Main.java : point d'entrée du programme, gère la saisie utilisateur et l'affichage

## Prérequis

- Java JDK 21
- IntelliJ IDEA

## Lancer le projet

1. Cloner le dépôt
2. Ouvrir le projet dans IntelliJ
3. Exécuter la classe Main

## Exemple d'exécution

Trouve le nombre entre 1 et 100. Tu as 10 essais !

Essai 1/10 - Ton nombre : 50 -> Trop petit !
Essai 2/10 - Ton nombre : 75 -> Trop grand !
Essai 3/10 - Ton nombre : 63 -> Gagné !

Bravo ! Trouvé en 3 essai(s) !
